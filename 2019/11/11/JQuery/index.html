JQuery

概念

一个javaScript的框架。简化js的开发

转换

  1. js -> jq: $(js对象)
  2. jq -> js: jq对象[index]或者jq对象.get(index)

    选择器

  3. 入口函数
    $(function(){})相当于windows.onload # onload只能执行一次
  4. 基本选择器
    1. 标签选择器
      • $(“html标签名”)
    2. id选择器
      • $(“#id”)
    3. 类选择器
      • $(“.class属性”)
    4. 并集选择器
      • $(“选择器1,选择器2”)
  5. 层级选择器
    1. 后代选择器
      • $(“A B”) A元素下的所有B元素
    2. 子选择器
      • $(“A > B”) A元素下的所有B子元素
  6. 属性选择器
    1. 属性名称选择器
      • $(“A[属性名]”)
    2. 属性选择器
      • $(“A[属性名=’值’]”)
    3. 符合属性选择器
      • $(“A[属性名=’值’][]…”)
  7. 过滤选择器
    1. 首元素选择器
      • :first
    2. 尾元素选择器
      • :last
    3. 非元素选择器
      • :not(selecter) 不包括指定内容的元素
    4. 偶数选择器
      • :even 偶数,从0开始计数
    5. 奇数选择器
      • :odd 奇数，从0开始计数
    6. 等于索引选择器
      • :eq(index) 指定索引元素
    7. 大于索引选择器
      • :gt(index)
    8. 小于索引选择器
      • :lt(index)
    9. 标题选择器
      • :header 获得标题(h1~h6)元素
  8. 表单过滤选择器
    1. 可用元素选择器
      • :enabled 获得可用元素
    2. 不可用元素选择器
      • :disabled 获得不可用元素
    3. 选中选择器
      • :checked 获得单选/复选框选中的元素
    4. 选中选择器
      • :selected 获得下拉框选中的元素

        DOM操作

  9. 内容操作
    1. html() 标签体内容(包含子标签体)
    2. text() 标签体纯文本内容
    3. val() 获取/设置元素的value属性值
  10. 属性操作
    1. 通用属性操作
      1. attr(name,val)
      2. removeAttr(name)
      3. prop(name,val)
      4. removeProp(name)
      5. css(属性,值)

        区别：如果操作是元素的固有属性，则建议使用prop。自定义属性则用attr。

    2. 对class属性操作
      1. addClass(class) 与prop(“class”,class)类似
      2. removeClass(class)
      3. toggleClass(class) 切换class(如果存在(不存在)就删除(添加)一个类)
  11. CRUD操作
    1. append() 父元素将子元素添加到末尾
      对象一.append(对象二): 将对象二添加到对象一元素内部，并且在末尾
    2. prepend() 在开头
    3. appendTo() 与1相反
    4. prependTo() 与2相反
    5. after() 添加元素到元素后边(兄弟关系)
      对象一.after(对象二) 将对象二添加到对象一后边
    6. before() 前边
    7. insertAfter() 与5相反
    8. insertBefore() 与6相反
    9. remove() 移除元素
    10. empty() 清空所有后代元素

高级

  1. 动画
    1. 默认显示和隐藏方式
      1. show([speed,[easing],[fn]])
      2. hide([speed,[easing],[fn]])
      3. toggle([speed,[easing],[fn]])
    2. 滑动显示和隐藏方式
      1. slideDown([speed,[easing],[fn]])
      2. slideUp([speed,[easing],[fn]])
      3. slideToggle([speed,[easing],[fn]])
    3. 淡入淡出
      1. fadeIn([speed,[easing],[fn]])
      2. fadeOut([speed,[easing],[fn]])
      3. fadeToggle([speed,[easing],[fn]])
    4. 参数
      1. speed 动画速度 (slow, normal, fast)或者动画时长毫秒值
      2. easing 切换效果 (swing, liner)(swing先慢再快再慢,liner匀速)
      3. fn 再动画完成时执行的函数，每个元素执行一次。
  2. 遍历
    1. jq对象.each(function(index, elemenvt))
    2. $.each(Object, callback )
    3. for..of: 3.0之后提供
    4. return false/return true = break/continue
  3. 事件绑定
    1. jquery标准的绑定方式
      js对象.事件方法(callback)
    2. on绑定事件/off解除绑定
      jq对象.on(“事件名称”,callback)
      jq对象.off(“事件名称”)
    3. 事件切换：toggle
      jq对象.toggle(fn1,fn2…)
  4. 插件
  • 增强JQuery的功能
  • $.fn.extend(object) 添加jQuery功能
    • $.fn.extend({check:callback})
  • $.extend(object) 增强jQuery对象自身的功能
    • $/JQuery
