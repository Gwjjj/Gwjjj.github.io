AJAX

概念

ASynchronous JavaScript And XML 异步的js和xml
无需加载整个页面的情况下能够更新部分网页技术

实现方式

  1. 异步和同步
    客户端和服务器相互通信的基础上
    同步：客户端必须等待服务器的响应，等待期间不能做其他操作
    异步：客户端不需要等待服务器的响应

  2. 实现方式

    1. 原生的js实现方式

    2. JQuery实现方式

      1. $.ajax()

      2. $.get()

      3. $.post()
